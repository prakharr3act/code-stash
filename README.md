# 📦 code-stash

A personal "second brain" for my web development journey. This repository serves as a central storage for daily experiments, historical projects, and reusable code snippets.

---

## 🚀 What's Inside?
This repo contains a mix of **HTML**, **CSS**, and **JavaScript** logic, organized by date and topic:

* **/2026**: Daily practice, UI components, and new logic.
* **/Archive**: Code and projects from 2+ months ago.
* **/Snippets**: Reusable CSS tricks and JS helper functions.

## 🛠️ Tech Stack
* **HTML5** (Semantic structure)
* **CSS3** (Flexbox, Grid, Animations)
* **JavaScript** (ES6+, DOM manipulation)

## 📌 Goals
1.  **Storage**: Keep all my work in one searchable place.
2.  **Tracking**: See how my coding style evolves over time.
3.  **Efficiency**: Quickly grab old code to use in new projects.
